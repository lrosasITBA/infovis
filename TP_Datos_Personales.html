1. Presentación

Mi objetivo de este trabajo fue darle un seguimiento a mi peso y masa grasa.
Como herramientas de recolección de datos, utilize FatSecret para anotar mis calorias y la App Fitness para contar las calorias quemadas por movimiento.
También lleve registro de actividades físicas como el gimnasio en la App de Notas.

2. Metodología

Todos los días se anotó la cantidad de calorías consumidas, calorías quemadas por movimiento y el cálculo de calorías quemadas en actividades físicas.
Se tomo como parametro de tasa metabólica basal (BMR): 2000kcal/día. El estandar promedio establecido para un hombre.
Para las mediciones de porcentaje de grasa, se utilizo el método "U.S. Navy". Calculado como: %BF = 495 / (1.0324 − 0.19077 × log10(waist − neck) + 0.15456 × log10(height)) − 450.

4. Explicación de errores

Si bien es una sorpresa agradable haber bajado kilos de más, este ánalisis prueba la falla del modelo.
Algunas de las posibles teorías para esto son:
-Se sobredimensiono al momento de contar las calorías consumidas.
-Se tomo una baja tasa metabólica basal.
-No se tomó en cuenta formas poco intensivas de actividad física, como tareas del hogar y movimientos de objetos pesados.
-Se hizo una incorrecta traducción de proteínas a kcal. Como las proteínas cumplen otras funciones como síntesis de aminoácidos, no es correcto relacionar todo el consumo de este macronutriente como fuente de calorías.

Adiconalmente, muestro una descomposición de los macronutrientes concumidos. Al tratarse de una dieta alta en proteína, este ultimo punto toma un peso considerable.
